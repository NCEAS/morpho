Table Editor

A Table is a regular collection of columns and rows containing data, e.g., the typical format used when data are stored in an ASCII file or spreadsheet. Use the Table Editor to enter and modify metadata describing the structure and fields within a Table.

The left side of the screen shows general information about the table. To edit this information click the 'Edit Table Description' button. The right box shows all of the defined fields (also know as "attributes" or "variables") within the table. To edit the description for any of these fields, either double click on the field of interest or click the 'Edit Attributes' button.
