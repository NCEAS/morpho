Table Editor

 

A Table is a regular collection of columns and rows containing data, e.g., the typical format used when data are stored in an ASCII file or spreadsheet. Use the Table Editor to enter and modify metadata describing the structure and fields within a Table. (You can open the editor by clicking the "edit" button at the top right corner of the table documentation panel when viewing a data package)

The left side of the screen shows general information about the table. Selecting one of the parameters on the left will cause the values for that parameter to be shown in the right frame. The right frame shows all of the defined fields (also know as "attributes" or "variables") within the table. You can directly click and edit the contents of any of these fields.
