The Package Editor

The main screen of the Package Editor


The table editor. The left side shows general information about the table. To edit this information click the 'Edit Table Description' button. The right box shows all of the defined fields (Attributes) within the table. To edit the description for any of these fields, either double click on the field of interest or click the 'Edit Attributes' button.


The first screen of the New Description Wizard.


This is the list of new descriptions that can be added to your package.


The editor allows you to add information to your new description.


Click the finish button to complete the description addition process.
